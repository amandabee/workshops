You're going to discover fairly quickly that if you don't have an "index.html" in your repository, you just get a 404 File Not Found error when you try to view your repository.

There are a few possible solutions:

  • create a file called "index.html" or,
  • explicitly name the file you want to see, in your URL.

For example, in my workshops repository, the gh-pages branch does not include a file called "index.html" so I get an error when I visit http://amandabee.github.io/workshops/. I do have a file there called "urls.html", however, so this works just fine: http://amandabee.github.io/workshops/urls.html
